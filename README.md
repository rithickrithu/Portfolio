## 🚀 How to Host on GitHub Pages

1. **Create a GitHub Repository:**
   - Go to [GitHub](https://github.com/new) and create a new public repository (e.g., `portfolio`).

2. **Push the Code:**
   Open your terminal in this folder (`C:\Users\rithick.kv\.gemini\antigravity\scratch\portfolio`) and run:
   ```bash
   git init
   git add .
   git commit -m "Initial commit - Portfolio"
   git branch -M main
   git remote add origin https://github.com/rithickrithu/<YOUR_REPO_NAME>.git
   git push -u origin main
   ```

3. **Enable GitHub Pages:**
   - Go to your repository on GitHub.
   - Click on **Settings** > **Pages** (under Code and automation on the left sidebar).
   - Under **Build and deployment**, set the Source to **Deploy from a branch**.
   - Select `main` branch and `/ (root)` folder, then click **Save**.
   - Within a few minutes, your portfolio will be live at `https://rithickrithu.github.io/<YOUR_REPO_NAME>/`!
